Independent Data Integrity Validation Framework

Business Vision

Create an independent validation framework that verifies dashboard accuracy and business rules regardless of whether dashboards are built using AI, Power BI, or other technologies.

1. Scheduled Validation
Run on demand or on a schedule.
2. Collect Dashboard Values
Python + Playwright securely opens the dashboard and captures values exactly as users see them.
3. Collect Source Data
Read Smartsheet and other approved source systems.
4. Normalize Data
Standardize dates, currencies, blanks, text and number formats.
5. Field-by-Field Validation
  • Missing records
  • Incorrect values
  • Truncated text
  • Reworded descriptions
  • Status/date mismatches
  • Duplicate records
6. Business Rule Validation
  • Quarterly budget request
  • Cumulative approved spend
  • Actual spend
  • Remaining budget
  • Quarter-over-quarter comparisons
  • Timeline milestone validation (First Registration, First Patient Dosed, Future Assumptions)
7. Generate Validation Report
Summarize discrepancies, quality metrics and impacted projects.
8. Notify Team
Email, Teams or Jira ticket when validation fails.

Phase Roadmap

  1. Phase 1: Validate current dashboard data.
  2. Phase 2: Automate validation using Smartsheet APIs and business rules.
  3. Phase 3: Reuse the framework for a financial governance dashboard. Claude may optionally generate narrative summaries, but validation remains independent.

Key Differentiators

  • Independent of AI implementation
  • Reusable across dashboards
  • Validates both data integrity and business logic
  • Scalable enterprise architecture
  • Produces auditable validation reports
